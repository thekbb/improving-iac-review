## Run Locally

1. `pnpm install`
1. `pnpm dev`
1. Open [http://localhost:3030](http://localhost:3030)

## Operate The Presentation

1. Start the dev server.
1. Use the spacebar or right arrow to advance.
1. Use the left arrow to go back.
1. Press `P` to open Presenter Mode.
1. Edit `slides.md` to change content.
